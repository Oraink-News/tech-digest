# Oraink Tech Digest

Public tech and internet daily/weekly digests by Oraink, curated from quality blogs, newsletters, public accounts, and social feeds.

## Archive

- Daily digests will be published under `daily/`.
- Weekly digests will be published under `weekly/`.
- Machine-readable metadata will be published in `manifest.json`.
